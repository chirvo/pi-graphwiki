# pi-graphwiki

**Self-curating wiki from graphify knowledge graphs.**

Reads `graphify-out/graph.json` (plus community analysis) and generates a browsable wiki of:

- **Concept pages** — one per graph node, with synthesized prose describing what it is, what connects to it, and why it matters
- **Community overviews** — per-cluster summaries with member inventory and topological role
- **Comparison pages** — auto-detected structural rivals and cross-community bridges
- **Glossary index** — alphabetized A–Z with one-line descriptions

All wiki files are indexed into **mempalace** (via `ctx_index`) so they're semantically searchable from any session.

## Usage

```bash
# Generate wiki from an existing graphify output
graphwiki ./graphify-out

# Generate + index into mempalace in one step
graphwiki ./graphify-out --index

# Quick: one-shot from current ./graphify-out
graphwiki

# Re-index without regenerating
graphwiki --reindex
```

## What you get

```
graphify-out/
├── graph.json                    # (existing) from graphify
├── GRAPH_REPORT.md               # (existing)
├── graph.html                    # (existing)
└── wiki/                         # <-- generated by pi-graphwiki
    ├── INDEX.md                  # Full glossary A–Z
    ├── nodes/
    │   ├── transformer.md        # Concept page: Transformer architecture
    │   ├── attention.md          # Concept page: Attention mechanism
    │   └── ... (per node)
    ├── communities/
    │   ├── _community-0.md       # Overview: "Attention Mechanism" cluster
    │   ├── _community-1.md       # Overview: "Training Pipeline" cluster
    │   └── ...
    └── comparisons/
        └── structural-rivals.md  # Auto-detected comparisons
```

## Integration with the Workglow

```
graphify --update → graphify-out/graph.json
       ↓
pi-graphwiki    → graphify-out/wiki/  + ctx_index → mempalace FTS5
       ↓
ctx_search("transformer architecture")
      → returns wiki prose + graph structure
```

## Installation

As a pi package:

```json
// settings.json
{
  "packages": ["git:github.com/chirvo/pi-graphwiki"]
}
```

Or extend from `~/.pi/agent/extensions/pi-graphwiki/`.
